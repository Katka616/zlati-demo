# Design System — Zlatí Řemeslníci

Celý vizuálny systém je definovaný v `src/app/globals.css` ako **CSS custom properties**. Žiadny Tailwind, žiadne CSS modules. Téma sa prepína cez atribút `data-theme="dark"|"light"` na `<html>` elemente.

## Layout tokeny

| Token | Hodnota | Použitie |
|---|---|---|
| `--radius` | 14px | Štandardné karty, modaly |
| `--radius-sm` | 8px | Malé tlačidlá, inputy |
| `--radius-lg` | 20px | Veľké karty, bottom sheet |
| `--max-width` | 500px | Maximálna šírka mobilného layoutu |
| `--tab-bar-height` | 67px | Výška bottom tab baru |
| `--transition` | 0.25s cubic-bezier(0.4, 0, 0.2, 1) | Animácie |

## Písma

| Font | Použitie | Import |
|---|---|---|
| **Cinzel** | Nadpisy (`h1`–`h3`), branding | Google Fonts |
| **Montserrat** | Telo textu, UI elementy | Google Fonts |

```css
/* Príklad použitia */
h1 {
  font-family: 'Cinzel', serif;
  color: var(--gold);
}

p {
  font-family: 'Montserrat', sans-serif;
  color: var(--text-primary);
}
```

## Témy

### Dark (luxusná čierna & zlatá) — default

```
--bg-primary:   #050505
--bg-card:      rgba(255,255,255,0.08)
--text-primary: #ffffff
--border:       rgba(255,255,255,0.10)
```

### Light (elegantná krémová & zlatá)

```
--bg-primary:   #FAFAF7
--bg-card:      #FFFFFF
--text-primary: #1A1A1A
--border:       #E8E2D6
```

> Téma sa nastavuje cez `document.documentElement.setAttribute('data-theme', 'dark')`.
> V Storybooku sa prepína cez toolbar vpravo hore (☀️ / 🌑).

---

## Pravidlo kontrastu (POVINNÉ)

- Text **nikdy** `var(--g5)` alebo svetlejší na bielom pozadí
- Minimum: `var(--g4)`, ideálne `var(--dark)`
- Sekundárny text: minimum `var(--g4)` alebo `#4B5563`
- Na tmavom pozadí: minimum `rgba(255,255,255,0.75)`
- Labely: vždy `fontWeight: 600+`
